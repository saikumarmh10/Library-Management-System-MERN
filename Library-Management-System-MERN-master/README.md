# 📚 Library Management System (MERN Stack)

A full-stack Library Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The application enables users to browse books, register/login securely, and manage library operations through an intuitive and responsive interface.

## 🚀 Live Demo

- **Frontend:** https://library-management-system-mern-nu.vercel.app
- **Backend API:** https://library-management-system-mern-xq32.onrender.com

---

## ✨ Features

### 👤 Authentication
- User Registration
- Secure Login
- JWT Authentication
- Password Encryption using bcrypt
- Role-based Access (Admin & Student)

### 📚 Library Management
- View Available Books
- Search Books
- Add New Books (Admin)
- Update Book Details
- Delete Books
- Book Issue & Return Management

### 🎨 User Interface
- Responsive Design
- Modern React UI
- Easy Navigation
- Mobile Friendly

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/saikumarmh10/Library-Management-System-MERN.git
```

### Install Backend

```bash
cd backend
npm install
```

### Install Frontend

```bash
cd ../frontend
npm install
```

---

## ▶️ Run the Project

### Backend

```bash
cd backend
npm start
```

### Frontend

```bash
cd frontend
npm start
```

---

## 🔐 Environment Variables

### Backend (.env)

```env
MONGO_URL=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Frontend (.env)

```env
REACT_APP_API_URL=https://library-management-system-mern-xq32.onrender.com
```
